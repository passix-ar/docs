Los **afiliados** te permiten medir cuántas ventas trae cada promotor, RRPP o canal, usando un link propio para cada uno.

![Gestión de afiliados](/img/panel/ev-affiliates.png)

## Para qué sirve

- Saber qué **RRPP / promotor** vendió más.
- Medir el rendimiento de cada **canal** (Instagram, WhatsApp, prensa).
- Liquidar comisiones a promotores con datos reales.

## Crear un afiliado

1. En el evento, andá a **Afiliados**.
2. **Crear afiliado**: nombre y (opcional) datos de contacto.
3. Passix genera un **link con seguimiento** para esa persona.
4. Compartí ese link con el promotor.

## Seguimiento

Por cada afiliado ves:

- **Ventas** atribuidas a su link.
- **Ingresos** generados.
- Cantidad de **entradas** vendidas.

## Diferencia con los códigos promo

- **Afiliado**: mide *quién trajo* la venta (atribución por link). No implica descuento.
- [Código promocional](/evento/codigos-promocionales/): aplica *descuento* o acceso. También sirve para atribuir por canal.

Podés usarlos combinados: link de afiliado + su propio código.

:::tip
Dale a cada RRPP su **link de afiliado**. Al cierre del evento tenés el ranking de ventas listo para liquidar comisiones.
:::
