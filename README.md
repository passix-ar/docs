# passix/docs

Documentación viva del proyecto Passix. Repo **privado**.

- **`DECISIONS.md`** — registro de decisiones (licencia, atribución, qué se publica, arquitectura, pagos). La fuente de verdad cuando hay dudas.
- **`ARCHITECTURE.md`** — cómo encajan las piezas (landing, Hi.Events, infra).
- **`ONBOARDING.md`** — cómo un dev levanta el workspace de cero.

Convención: toda decisión que cueste re-discutir va a `DECISIONS.md` con su número (D1, D2…) y, si abre un riesgo, queda anotado en la sección de riesgos (R1…).
